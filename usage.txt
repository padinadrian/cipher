Usage: cipher -m METHOD -p PASSWORD [INPUT_FILE] [OUTPUT_FILE]
Encrypt or decrypt text using simple ciphers
Example: cipher -m vigenere -p HELLOWORLD - ciphertext.txt

With no INPUT_FILE, or when INPUT_FILE is -, read standard input.
With no OUTPUT_FILE, or when OUTPUT_FILE is -, read standard input.

  -h    Print this help message and exit
  -v    Print extended version information and exit
  -m    Use encryption method METHOD
        Supported options for METHOD: 'caesar', 'vigenere'
  -p    PASSWORD will be used as the cipher key
